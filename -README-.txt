README

Ceci est un programme permettant de compter les clics avec le module Turtle, un module très peu performant et ayant très peu de fonctions comparé à d'autres.
C'est un petit projet que j'ai fait dans le cadre du divertissement pour me défier.

Ce programme consiste en un compteur qui s'affiche à l'écran et une image affichée en dessous qui change en fonction du nombre de clics. 
Cliquer sur l'image comptera 1 clic au début. 
Le compteur rajoute automatiquement des clics au fur et à mesure.

NECESSAIRE :

- le programme est le fichier "pokeballclicker.py"
- importer le module turtle à partir de l'invite de commande, avec la commande : pip install turtle
- importer toutes les images inclues dans le dossier
